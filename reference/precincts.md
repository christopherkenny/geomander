# precincts

This data contains the election districts (or precincts) for Rockland
County NY, with geographies simplified to allow for better examples.

## Usage

``` r
data("precincts")
```

## Format

An sf dataframe with 278 observations

## References

https://www.rocklandgis.com/portal/apps/sites/#/data/datasets/2d91f9db816c48318848ad66eb1a18e9

``` r
data('precincts')
```
