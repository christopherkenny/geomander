# List Available VEST States

List Available VEST States

## Usage

``` r
vest_states(year)
```

## Arguments

- year:

  Year to inspect in the VEST Dataverse collection.

## Value

character vector of state abbreviations available for that year

## Examples

``` r
if (FALSE) { # \dontrun{
# Requires Dataverse API
vest_states(2020)
} # }
```
