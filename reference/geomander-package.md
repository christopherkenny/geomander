# Geographic Tools for Studying Gerrymandering

A compilation of tools to complete common tasks for studying
gerrymandering. This focuses on the geographic tool side of common
problems, such as linking different levels of spatial units or
estimating how to break up units. Functions exist for creating
redistricting-focused data for the US.
