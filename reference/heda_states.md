# List Available HEDA States

List Available HEDA States

## Usage

``` r
heda_states()
```

## Value

character vector of lowercase state abbreviations with packaged HEDA
support.

## Examples

``` r
heda_states()
#>  [1] "tx" "ks" "ct" "ga" "mn" "md" "wa" "nd" "co" "wy" "nv" "nh" "vt" "nc" "nm"
#> [16] "ny" "ak" "oh" "hi" "ms" "de" "id" "wi" "in" "ne" "fl" "tn" "ma" "sc" "mi"
#> [31] "mo" "al" "az" "nj" "ca" "il" "ia" "pa" "la" "sd" "ok"
```
